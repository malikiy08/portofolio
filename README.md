## 🚀 Featured Projects

### 1. Social & News Media Crawler Engine
**Period:** 2023
**Role:** Backend Engineer
**Tech Stack:** Node.js, Puppeteer, Redis, Elasticsearch, Neo4j, Internal ML API

An intelligent data collection and analysis system that scrapes social media (Twitter, Facebook) and news platforms, processes content through ML pipelines for sentiment analysis and keyword extraction, then stores searchable data in Elasticsearch and relationship graphs in Neo4j for network visualization.

**Key Achievements:**
- ⚡ Processes **thousands of tweets per hour** with ML-powered analysis
- 🤖 Integration with **ML API** for sentiment analysis and keyword extraction
- 🔍 **Elasticsearch** for full-text search and analytics capabilities
- 🕸️ **Neo4j graph database** for relationship visualization (tweets, replies, retweets, mentions)
- 📊 Interactive dashboard with keyword search and network diagram visualization
- 🛡️ Anti-detection mechanisms and intelligent rate limiting
